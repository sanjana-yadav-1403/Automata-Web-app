Introduction

A pushdown automaton (PDA) is a finite automaton with an additional stack, which provides additional memory to store information. PDAs are used to recognize context-free languages and are more powerful than finite automata.

Components of a Pushdown Automaton

A pushdown automaton consists of the following components:

  • Finite set of states
  • Input alphabet
  • Stack alphabet
  • Transition function
  • Initial state
  • Stack bottom symbol
  • Set of accepting states
